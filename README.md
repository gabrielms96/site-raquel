# Psicóloga Raquel Pontifice

Site institucional desenvolvido para apresentação profissional de uma psicóloga clínica, com foco em presença digital e facilitação de contato com pacientes.

A proposta do projeto é funcionar como um cartão de visita online, reunindo informações essenciais de forma simples, clara e acessível.

## Acesse o site:  
https://www.psiraquelpontifice.com.br/

## Objetivo

Criar uma landing page leve e responsiva que permita:

- Apresentação profissional
- Informações sobre a abordagem terapêutica (TCC)
- Contato direto via WhatsApp
- Acesso ao Instagram profissional
- Formulário de pré-atendimento (Google Forms)
- Informações sobre atendimentos online e presenciais

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- Google Forms (integração externa)
- WhatsApp API (link direto)

## Funcionalidades

- Layout responsivo para mobile e desktop
- Navegação por seções (single page)
- Botões de contato com abertura direta no WhatsApp
- Link para formulário de pré-agendamento
- Estrutura simples e de fácil manutenção
- Deploy estático (GitHub Pages / Netlify)

Este projeto foi desenvolvido com foco em simplicidade, performance e facilidade de manutenção, sem uso de frameworks ou bibliotecas externas.

A proposta é servir como uma solução prática para presença digital profissional.
